## Getting Started

### Creating an Account

1. **Visit** [https://edify.ai](https://edify.ai)
2. **Click** "Sign Up" or "Continue with Google"
3. **Complete your profile:**
   - Name and email
   - Learning goals (e.g., "Master machine learning fundamentals")
   - Current subjects of interest
   - Skill level (beginner, intermediate, advanced)

4. **Verify your email** (check your inbox)
5. **Choose your subscription tier:**
   - **Free:** 50 questions/month, basic features
   - **Premium:** Unlimited questions, advanced analytics
   - **Enterprise:** Custom features, dedicated support

### First Login

After signing in, you'll see:
- **Learning Dashboard:** Your personalized home base
- **Quick Start Tutorial:** 5-minute interactive walkthrough
- **Recommended Topics:** Based on your profile

### Step 1: Ask a Question

Navigate to the **Chat** interface and ask your first question:

**Example questions:**

EDIFY provides:
- **Personalized answer** adapted to your skill level

## Tips for Effective Learning

### Best Practices

**1. Be Specific in Questions**
```
âŒ "Tell me about Python"
âœ… "How do Python decorators work with examples?"
```

**2. Build on Previous Conversations**
```
Session 1: Learn basics
Session 2: Explore applications
Session 3: Implement projects
```

**3. Use Follow-Up Questions**
```
Initial: "Explain gradient descent"
Follow-up: "Can you show the mathematical formulation?"
Follow-up: "What are common variants?"
```

**4. Set Learning Goals**
- Define clear objectives (e.g., "Build a CNN classifier")
- Break into smaller milestones
- Track progress weekly

**5. Review Regularly**
- Revisit concepts after 1 day, 1 week, 1 month
- Use "Review" feature in dashboard
- Test understanding with practice problems

### Common Mistakes to Avoid

**âŒ Asking Too Broadly**
"Tell me everything about AI" â†’ Too overwhelming

**âŒ Not Following Up**
Get one answer and move on â†’ Shallow learning

**âŒ Ignoring Prerequisites**
Jump to advanced topics â†’ Confusion and frustration

**âŒ Passive Reading**
Just read responses â†’ Poor retention

**âœ… Active Learning Approach**
1. Ask question
2. Understand explanation
3. Ask clarifying questions
4. Apply to examples
5. Implement in code
6. Review after time

### Learning Strategies

**Spaced Repetition:**
- Review concepts at increasing intervals
- Use EDIFY's reminder feature
- Track what needs review

**Feynman Technique:**
1. Learn concept from EDIFY
2. Explain in your own words
3. Identify gaps in understanding
4. Ask EDIFY to clarify gaps

**Project-Based Learning:**
1. Define project goal
2. Identify concepts needed
3. Learn each concept
4. Apply in project
5. Ask EDIFY for guidance

## For Educators

### Educator Features

**Access:** Request educator role via Settings > Request Upgrade

**Capabilities:**
- Upload custom course materials
- Create class groups
- View class analytics
- Set learning objectives for students
- Monitor student progress
- Generate progress reports

### Uploading Course Content

1. Navigate to **Content** > **Upload**
2. Select files (PDF, DOCX, TXT)
3. Add metadata:
   ```
   Title: "Introduction to Machine Learning"
   Subject: "Machine Learning"
   Difficulty: "Intermediate"
   Tags: ["supervised learning", "neural networks"]
   ```
4. Wait for processing (3-5 minutes)
5. Content becomes searchable for your students

### Viewing Class Analytics

**Dashboard** > **Classes** > Select Class

**Metrics Available:**
- Average progress per student
- Common difficulties (topics where students struggle)
- Engagement metrics
- Popular topics
- Retention rates
- Time-to-mastery averages

**Export Reports:**
- PDF summary reports
- CSV data for custom analysis
- Individual student progress reports

### Best Practices for Educators

1. **Supplement, Don't Replace:** Use EDIFY alongside traditional teaching
2. **Upload Materials:** Add your own slides, notes, textbooks
3. **Monitor Trends:** Identify where students struggle collectively
4. **Encourage Active Use:** Set expectations for engagement
5. **Review Analytics:** Use data to adjust teaching approach

## Privacy & Data

### What EDIFY Tracks

**To improve your learning:**
- Questions you ask
- Topics you explore
- Time spent per concept
- Progress over time

**Never tracked:**
- Private conversations (unless you share)
- Personal browsing outside EDIFY
- Sensitive personal information

### Data Control

**Your Rights:**
- View all your data: Settings > Privacy > Download Data
- Delete your data: Settings > Privacy > Delete Account
- Opt-out of analytics: Settings > Privacy > Analytics (keeps basic features)

**GDPR Compliant:**
- Data encrypted at rest and in transit
- 90-day retention for logs
- Right to be forgotten
