# GUIRA Demo Videos

This directory contains demonstration videos showcasing GUIRA's capabilities.

## Video Requirements

### 1. System Demo
**Filename:** `guira-demo.mp4`  
**Duration:** 60-90 seconds  
**Content:**
- Fire detection in action
- Smoke pattern analysis
- Real-time risk map generation
- Alert notification flow
- Multi-model integration showcase

### 2. Field Testing Footage
**Filename:** `field-test-demo.mp4`  
**Duration:** 2-3 minutes  
**Content:**
- Actual field deployment
- Detection accuracy demonstration
- Prediction validation
- Community interaction

### 3. Tutorial Video
**Filename:** `tutorial-setup.mp4`  
**Duration:** 5-10 minutes  
**Content:**
- Installation walkthrough
- Configuration guide
- First detection demo
- Troubleshooting tips

## Video Specifications

- **Format:** MP4 (H.264 codec)
- **Resolution:** 1920x1080 minimum
- **Frame Rate:** 30 FPS
- **Audio:** Optional narration
- **Subtitles:** English (SRT file)
- **Size:** Under 100MB (compressed)
