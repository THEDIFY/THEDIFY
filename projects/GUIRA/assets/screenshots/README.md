## Screenshot Guidelines

- **Resolution:** Minimum 1920x1080
- **Format:** PNG (preferred) or JPEG
- **Content:** Show real or simulated GUIRA functionality
- **Annotations:** Include labels, legends, timestamps
- **Privacy:** No personally identifiable information
- **Quality:** High resolution, clear visibility
