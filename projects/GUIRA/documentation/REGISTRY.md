# Dataset Registry

This document provides an overview of all datasets used in this project. Each dataset has a unique identifier used in configuration files and scripts.

| Identifier | Dataset Name | Source | Description |
| :--- | :--- | :--- | :--- |
| `flame_rgb` | FLAME (UAV Wildfire) | [GitHub](https://github.com/AlirezaShamsoshoara/Fire-Detection-UAV-Aerial-Image-Classification-Segmentation-UnmannedAerialVehicle) | RGB frames and pixel-level fire masks from UAV imagery. |
| `flame_rgb_simplified` | Sunnyiisc Fire-Detection from FLAME | [GitHub](https://github.com/sunnyiisc/Fire-Detection-from-FLAME-Dataset) | Simplified version of FLAME with ready-to-run code. |
| `flame2_rgb_ir` | Flame_2 Dataset | [GitHub](https://github.com/xiwenc1/Flame_2_dataset) | Paired RGB and thermal infrared (IR) drone data for fire detection. |
| `sfgdn_fire` | SFGDN Flame-detection | [GitHub](https://github.com/mi-luo/Flame-detection) | Salience-guided model and dataset for UAV remote sensing flame detection. |
| `flame3_thermal` | FLAME 3 | [arXiv](https://arxiv.org/abs/2412.02831) | Advanced dataset with radiometric thermal imagery. |
| `wit_uas_thermal` | WIT-UAS | [arXiv](https://arxiv.org/abs/2312.09159) | Long-wave infrared (LWIR) dataset for detecting assets in fire scenes. |
| `waid_fauna` | WAID (Wildlife Aerial Images from Drones) | [GitHub](https://github.com/xiaohuicui/WAID) | ~14k drone images across six species with bounding-box annotations. |
| `kaggle_fauna` | Kaggle Wildlife Aerial Imagery | [Kaggle](https://www.kaggle.com/datasets/sugamg/wildlife-aerial-imagery-dataset) | Aerial drone wildlife dataset. |
| `awir_fauna` | AWIR (Aerial Wildlife Image Repository) | [PMC](https://pmc.ncbi.nlm.nih.gov/articles/PMC11265857/) | Community-curated repository for UAV wildlife monitoring. |
| `neon_canopy` | DeepForest NEON | [DeepForest](https://deepforest.readthedocs.io/en/latest/) | Pre-trained models on NEON data for tree canopy detection. |
| `isaid_aerial` | iSAID | [Website](https://captain-whu.github.io/iSAID/) | A large-scale dataset for instance segmentation in aerial images. |
